Simple Login API Documentation

Overview

The Frappe Framework provides a robust and flexible web application framework built with Python. This documentation covers the essential login API endpoints for user authentication and session management.

Login Endpoint

POST
/api/method/login

Request Parameters

Parameter Type Required Description
usr string Yes Username
pwd string Yes Password

Request Example

curl -X POST https://your-frappe-instance.com/api/method/login \
   -H 'Content-Type: application/json' \
   -H 'Accept: application/json' \
   -d '{{ '{' }}"usr":"Administrator","pwd":"admin"{{ '}' }}'

Response

Success Response (200)
{{ '{' }}
 "home_page": "/desk",
 "full_name": "Administrator",
 "message": "Logged in"
{{ '}' }}

Cookie: sid (send this to authenticate future requests). Expires in three days.

sid=05d8d46aaebff1c87a90f570a3ff1c0f570a3ff1c87a90f56bacd4;
   path=/;
   domain=.your-frappe-instance.com;
   Expires=Sat, 29 Sep 2018 00:59:54 GMT;
Error Response (401)
Wrong password or username.

Logout Endpoint

GET
/api/method/logout

Request Example

curl -X GET https://your-frappe-instance.com/api/method/logout

Response

Success Response (200)
{{ '{' }}{{ '}' }}

Get Current User

GET
/api/method/frappe.auth.get_logged_user

Request Example

curl -X GET https://your-frappe-instance.com/api/method/frappe.auth.get_logged_user

Response

Success Response (200)
{{ '{' }}
 "message": "Administrator"
{{ '}' }}

Implementation Notes

  • Content-Type: Use application/x-www-form-urlencoded for login requests
  • Session Management: Frappe uses server-side sessions with cookies for authentication
  • Cookie Authentication: The sid cookie must be included in subsequent requests
  • Security: Passwords are hashed and never stored in plain text
  • Error Handling: Handle 401 errors for invalid credentials

Code Examples

JavaScript/Node.js

const axios = require('axios');
  
  // Login
  const loginResponse = await axios.post('https://your-instance.com/api/method/login', {{ '{' }}
    usr: 'Administrator',
    pwd: 'admin'
  {{ '}' }}, {{ '{' }}
    headers: {{ '{' }}
      'Content-Type': 'application/json',
      'Accept': 'application/json'
    {{ '}' }}
  {{ '}' }});
  
  // Store session cookie
  const sessionCookie = loginResponse.headers['set-cookie'];
  
  // Make authenticated requests
  const userResponse = await axios.get('https://your-instance.com/api/method/frappe.auth.get_logged_user', {{ '{' }}
    headers: {{ '{' }}
      'Cookie': sessionCookie
    {{ '}' }}
  {{ '}' }});

Python

import requests
  
  # Login
  login_data = {{ '{' }}
      'usr': 'Administrator',
      'pwd': 'admin'
  {{ '}' }}
  
  response = requests.post('https://your-instance.com/api/method/login', 
                          json=login_data,
                          headers={{ '{' }}'Accept': 'application/json'{{ '}' }})
  session_cookies = response.cookies
  
  # Make authenticated requests
  user_response = requests.get(
      'https://your-instance.com/api/method/frappe.auth.get_logged_user',
      cookies=session_cookies
  )
